# README #

This repository contains the source code to the App Pokemon.

### Live demo ###
[Pokemon App](http://joeloliveira.org/pokemon)

### How do I get set up? ###

* Install NodeJS
* Install ruby

### Run the following Script ###

* gem update --system
* gem install compass
* npm install
* bower install
* npm install -g grunt
* npm install -g grunt-cli
* npm install grunt --save-dev

### Run App ###

* grunt serve


### Run Unit Tests ###

* grunt test


### Release Application ###

* grunt build
